# CoachChess 2.4

## Neu

- Stockfish bewertet jeden weißen Zug.
- Anzeige von: Bester Zug, Sehr gut, Gut, Ungenauigkeit, Fehler oder Grober Fehler.
- Stellungsbewertung aus Sicht von Weiß.
- Beste Stockfish-Alternative und Verlust gegenüber dem besten Zug.
- Kurze Trainer-Erklärung.
- Deutsche Zugnotation:
  - S = Springer
  - L = Läufer
  - T = Turm
  - D = Dame
  - K = König

FEN bleibt unverändert, da FEN ein international festgelegtes technisches Format ist.
